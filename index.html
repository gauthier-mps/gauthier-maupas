PING

Projet EPITA

Durée : 3 semaines

Objectif : Créer un IDE

Langages : HTML, CSS, Javascript, Java

   Le projet PING est un projet en groupe de 5 ou nous devions développer un IDE (un environnement de développement) pour un client. Nous avons développé le backend de notre IDE en Java (langage imposé) et nous avons développé le front-end avec électron en html, css et javascript. Pour connecter notre back-end avec le front-end nous avons fait une API en Java.

Tiger

Projet EPITA

Durée : 7 semaines

Objectif : Créer un compilateur

Langages : C++, Bison, Re/Flex

   Le projet Tiger est un projet, en groupe de 4, de 3ème année à EPITA dont l'objectif est de créer un compilateur de Tiger en utilisant le langage C++. Pour ce projet, EPITA fournit une grande base de code qui doit être modifiée et complétée pour que le compilateur fonctionne. Chaque semaine, de nouvelles fonctionnalités devaient être implémentées. Au total, nous avons ajouté plus de 7 000 lignes de code.

42sh

Projet EPITA

Durée : 4 semaines

Objectif : Créer un shell POSIX

Langages : C

   Le projet 42sh est un projet emblématique d'EPITA en groupe de 4. Il s'agit d'un projet de développement en langage C où nous avons créé un shell similaire à Bash à partir de zéro. À la fin du projet, nous avons obtenu un shell fonctionnel qui permet à l'utilisateur d'exécuter toutes les commandes connues sous Unix. Nous avons également recodé certains des "builtins" tels que "echo" et "cd".

JWS (Java Web Services)

Projet EPITA

Durée : 1 semaines

Objectif : Créer une API REST

Langages / Frameworks : Java / Quarkus, Hibernate

   JWS est un projet en Java dont l'objectif était de créer le back-end d'un jeu Bomberman en Java. Ce projet a été très formateur, car il nous a permis d'apprendre comment fonctionne une API REST et comment elle est structurée.

Sudoku Solver

Projet EPITA

Durée : 8 semaines

Objectif : Création d'un OCR pour résoudre des sudoku

Langages : C

   Le projet OCR est le premier grand projet de groupe à l'EPITA. OCR Sudoku Solver est un logiciel qui prend en entrée une photo d'un sudoku, traite l'image pour la rendre exploitable, résout le sudoku et affiche l'image du sudoku avec la grille remplie. Ce projet nous a permis d'apprendre les techniques de traitement d'image, les bases d'un réseau de neurones, ainsi que le développement d'une interface graphique.

ZeSouk

Projet EPITA

Durée : 16 semaines

Objectif : Création d'un jeu 2D

Langages : C# / Unity

   Pendant notre deuxième semestre de première année, nous avons eu pour projet de créer un jeu vidéo en groupe de 4 avec Unity.
