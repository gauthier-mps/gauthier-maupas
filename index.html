Mes Projets

Tiger

Type de projet : école - groupe de 4

Durée : 7 semaines

Période : février - mars 2023

Objectif : Créer un compilateur

Langages : C++, Bison, Re/Flex

Description : Le projet Tiger est un projet de 3ème année à EPITA dont l'objectif est de créer un compilateur de Tiger en utilisant le langage C++. Pour ce projet, EPITA fournit une grande base de code qui doit être modifiée et complétée pour que le compilateur fonctionne. Chaque semaine, de nouvelles fonctionnalités devaient être implémentées. Au total, nous avons ajouté plus de 7 000 lignes de code.

42sh

Type de projet : école - groupe de 4

Durée : 4 semaines

Période : janvier 2023

Objectif : Créer un shell posix

Langages : C

Description : Le projet 42sh est un projet emblématique d'EPITA. Il s'agit d'un projet de développement en langage C où nous avons créé un shell similaire à Bash à partir de zéro. À la fin du projet, nous avons obtenu un shell fonctionnel qui permet à l'utilisateur d'exécuter toutes les commandes connues sous Unix. Nous avons également recodé certains des "builtins" tels que "echo" et "cd".

Piscine (C - Java - SQL - C++ - JS)

Durée : 2 semaines pour la piscine de C, 1 semaine pour les autres

Période : ING1 (2022/2023)

Objectif : Apprentissage intensif d'un langage

Langages : C, Java, SQL, C++, JS

Description : Les piscines sont des ateliers d'une ou deux semaines pendant lesquels nous apprenons un nouveau langage de programmation. Les horaires de ces ateliers sont de 8h à minuit. Nous avons des cours et travaux dirigés le matin et en début d'après-midi pour apprendre les spécificités du langage. Le soir, nous avons des exercices pratiques à valider pour réussir l'atelier. Chaque piscine se termine par un "rush", qui est un projet individuel de 24h ou 48h à réaliser dans le langage que nous venons d'apprendre.

Mini projets

Type de projet : école - individuel

Durée : 4 semaines - 1 semaine par projets

Période : novembre 2022

Objectif : Réaliser des projets court terme

OCR - Sudoku Solver

Type de projet : école - groupe de 4

Durée : 8 semaine

Période : octobre - novembre 2021

Objectif : Création d'un OCR pour résoudre des sudoku

Langages : C

Description : Le projet OCR est le premier grand projet de groupe à l'EPITA. OCR Sudoku Solver est un logiciel qui prend en entrée une photo d'un sudoku, traite l'image pour la rendre exploitable, résout le sudoku et affiche l'image du sudoku avec la grille remplie. Ce projet nous a permis d'apprendre les techniques de traitement d'image, les bases d'un réseau de neurones, ainsi que le développement d'une interface graphique.

Ze Souk

Type de projet : école - groupe de 4

Durée : 16 semaines

Période : février - juin 2021

Objectif : Création d'un jeu 2D
